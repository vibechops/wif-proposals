## Program shape (revised)

A **modular Google Flow activation** across India, **16 weeks (4 months)**. Option A is the college event tour. Option B adds script call, Filmathon, and Mumbai awards in the same window.

After events, the program moves to a **national script call**. WIF curates scripts, selects filmmakers, and (if funded) gives them Flow access to produce fully AI short films. Top films are screened at a **Mumbai awards / PR event**; jury awards **1st, 2nd, 3rd place**.

## Four modules

| # | Module | Summary |
|---|---|---|
| 1 | **Flow Workshop Events** | Combined student + professional event at colleges; Google expert demos Flow; networking; YouTube livestream |
| 2 | **Script Call** | National topic announced; open script submissions; WIF curates and selects |
| 3 | **AI Filmathon** | Selected filmmakers get WIF-funded Flow access; produce fully AI shorts in 7-14 days |
| 4 | **Awards Ceremony** | Mumbai PR event; jury; Top 3; winners get 1-year free Google Flow subscription |

Modules 1+2 can run without Module 3 if WIF does not want to fund credits. See `program-flow.md` for flowchart.

## Film rules (from Google)

- Fully **AI-generated** using Google Flow / Google AI tools
- **No live-action / real-life footage**
- Film must be **about AI**
- Must show the **positive side of AI**

## WIF credit budget (summary)

Google will not fund tokens. If WIF runs Module 3:

- **~₹3,000 per selected filmmaker** (Pro subscription + buffer)
- **50 filmmakers = ~₹1.5L** (~$1,800) for credits alone
- Full program (all modules): **₹13L-44L** depending on scale
