## 1. Screenwriters Residency Lab

### Overview

Genre-based, pan-India feature film development lab. 6 scripts, 6 mentors, residential workshop, festival tie-up. IP stays with writers.

### Key parameters

| Question | Answer |
|---|---|
| Open to | Indian citizens; original fiction feature film ideas |
| Languages | All Indian languages (English translation if required) |
| Round 1 submission | 2-page concept note (1,000-1,200 words), genre specified |
| Round 2 submission | 5-page detailed treatment |
| Final cohort | 6 writers, 1 script per mentor |
| Residential lab | 5 days, Kolkata / Darjeeling |
| Participants on site | 6 writers + 6 mentors + 4 core team = 16 pax |
| Festival tie-up | Part 2: 2-3 day industry showcase (TIFF India / Film Bazaar style) |
| IP | Writer retains full ownership; mandatory credit: "Developed at the WIF India Screenwriters Lab" |

### Mentors (proposed)

Varun Grover, Juhi Chaturvedi, Smita Singh, Payal Kapadia, Kiran Rao, Bhavani Iyer, Gauri Shinde, Sumit Roy, Seeta Menon, Pooja Surti (6 selected for final cohort).

### Timeline

| Phase | Window |
|---|---|
| Call for applications | October 2025 |
| Shortlist (50) | December 2025 |
| Treatment phase | Jan-Feb 2026 |
| Final selection (6) | March 2026 |
| Writers Lab Part 1 (residential) | May 2026 |
| Writers Lab Part 2 (festival) | August 2026 |

### Budget (internal)

₹1.175 Cr grand total (incl. 10% contingency and ₹35L WIF fee). Source: `../00-intake/Screenwriters lab - BUDGET - Sheet1-3.pdf`

### Dell branding

- Presenting Partner on all applications, comms, and festival materials
- "Supported by Dell" on writer pitch decks and festival submissions
- Optional: Dell representative at residential lab or festival showcase

## 3. Internship Program

### Overview

Paid 6-month internships placing young women in hands-on roles across the Indian film, TV, and OTT ecosystem. Goal: 100 women over 3 years.

### Key parameters

| Question | Answer |
|---|---|
| Duration per intern | 6 months |
| Cohort size (Year 1) | 35 interns |
| 3-year target | 100 women |
| Departments | Direction, Writing, Editing, Production, Art, Costume, Casting, Entertainment Law, OTT Programming, Cinematography, and more |
| Stipend | ₹20,000/month per intern |
| Support | Travel and accommodation (especially non-metro); host mentorship; certificates and recommendation letters |
| Closing event | Formal ceremony; certificates co-branded Dell + WIF; Dell rep invited to present |

### Budget (internal, annual)

| Line item | Estimate (INR) |
|---|---:|
| Stipends (35 interns x 6 months x ₹20K) | 50,00,000 |
| Support team (research, HR, placement) | 60,00,000 |
| Host mentor honorariums | 30,00,000 |
| Travel and accommodation support | 40,00,000 |
| Program management + M&E | 35,00,000 |
| Closing ceremony | 10,00,000 |
| Contingency (10%) | 22,50,000 |
| WIF India program fee | 52,50,000 |
| **Annual total** | **2,50,00,000 (~₹2.5 Cr)** |

Source: adapted from `../00-intake/Proposal for Kamla Mills Foundation - Internal .docx`

### Impact metrics (Year 3 target)

- 100+ internships completed
- 50+ production companies engaged
- Conversion tracking: interns to full-time roles
- Annual WIF India Gender Employment Report

### Dell branding

- Presenting Partner on all recruitment materials

- Featured in annual impact report with placement data
